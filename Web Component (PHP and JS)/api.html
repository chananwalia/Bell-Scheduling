API Access Instructions Instructions to pull current schedule data.

Pull from the following URL to receive text with current schedule information.

/

Symbols can be any-length text but when set should be under three characters. Timing is also an unrestricted string. For agenda items, the type is either "Regular Class", "Break/Lunch", "Assembly/Liturgy", or "Other". For announcements, the type is either "Standard", "Assembly/Liturgy", or "Other".
